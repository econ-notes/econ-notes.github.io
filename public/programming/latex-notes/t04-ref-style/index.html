1. Changes to the original specification file

The attached file is adaped from the public good offered by Shiro Takeda: https://github.com/ShiroTakeda/econ-bst.

Personally, I prefer to make the following changes to this specification file.

  • The year field:
    • Position: bst.year.position
    • Separation marks: bst.year.pre and bst.year.post
  • The number field:
    • Existence: bst.hide.number
    • Separation marks: bst.number.pre and bst.number.post
  • The title field:
    • Separation marks: bst.title.pre and bst.title.post
  • The author field:
    • Separation marks: bst.author.pre and bst.author.post
  • On other non-article types (e.g., book, incollection etc.)
    • Separation marks: bst.year.na.pre and bst.year.na.post
    • Enclosing and appearence of the title: bst.book.pre and bst.book.post

2. Add this bst file globally

  • Update the TeX filename database with the bash command texhash (run in terminal).

%% An entry is a single document in the reference. The "entry type" refers to
%% the type of document, such as an article or a book.

%% Field is the title, author, year, etc. specified in an entry.

%% is the argument of the "\cite" command.

    institution journal key month note number organization pages publisher
    school series title type url volume year

  { order.cited }

FUNCTION {last.period.comma.p}

  t #-1 #1 substring$ "." = 
  t #-1 #1 substring$ "," = or
  t #-1 #2 substring$ ". " = or
  t #-1 #2 substring$ ", " = or
  t #-1 #2 substring$ ".'" = or
  t #-1 #2 substring$ ",'" = or
  t #-1 #3 substring$ ".''" = or
  t #-1 #3 substring$ ",''" = or
  t #-1 #2 substring$ ".~" = or
  t #-1 #2 substring$ ",~" = or
  t #-1 #1 substring$ ":" = or 
  t #-1 #2 substring$ ": " = or
  t #-1 #2 substring$ ".}" = or
  t #-1 #2 substring$ ",}" = or
    { #1 'last.period.comma := }

  if$

{ 's :=

  s num.names$ 'numnames :=

    { bst.max.author.num.display 'numtrunc := }
    { #0 'numtrunc := }
  if$
  % Repeat if namesleft > 0 ---- Repeat from here
  { namesleft #0 > }
  {
    % Block A starts
    nameptr #1 =
      % A-1: If the author is the 1st author 
      { bst.author.name #0 =
        bst.author.name #1 = or
           { bst.first.name.initial #0 =

              { s nameptr "{f.~}{vv~}{ll}{, jj}" format.name$ 't := }
             if$
           }
        if$
      }
      % A-2: If the author is the second or later authors (not the 1st author).
      { bst.author.name #0 =
        bst.author.name #2 =
        or
           { bst.first.name.initial #0 = 
              { s nameptr "{ff~}{vv~}{ll}{, jj}" format.name$ 't := }

                { s nameptr "{vv~}{ll}{, f.}{, jj}" format.name$ 't := }
             if$
           }
        if$
      }
    if$
    % Block A ends
    
    % Block B starts
    nameptr #1 >
      % B-1: If the author is the second or later authors
      { nameptr #1 - numtrunc =
          % If nameptr - 1 = numtrunc
    { #1 'namesleft := 
      "others" 't := }
    'skip$
  if$
        % If numnames <= bst.max.author.num
        nameorder.temp "random" =
        bst.use.nameorder #0 = not and
          % If random ordering is used
          { bst.and.nameorder * t * }

       }
